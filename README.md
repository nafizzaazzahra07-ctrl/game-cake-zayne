# Game Cake Zayne 🎂

Game Cake Zayne merupakan permainan arcade 2D interaktif yang dikembangkan menggunakan framework GREENFOOT dan bahasa pemrograman JAVA. Game ini merupakan salah satu implementasi untuk memahami konsep OOP (Object Oriented Programming) yang berfokus memahami sifat inheritance, Polymorphism, dan Encapsulation. Dalam game ini terdapat aktor utama yang dapat dikontrol oleh Pengguna. Pengguna bisa mengendalikan karakter Zayne untuk mengumpulkan makanan/buah-buahan tanpa terlewat satu bagian dan mempertahankan HP demi mendapatkan skor tertinggi.

## 🎮 Fitur & Komponen Game
* Sistem Buah & Kue : Menggunakan kelas `Makanan` sebagai objek yang harus dikumpulkan pemain.
* Sistem Skor Otomatis: Menggunakan kelas `Counter` untuk menghitung skor secara real-time selama permainan berlangsung.
* Gameplay Responsif: Gerakan karakter yang lancar dan interaksi antar objek yang dinamis.

## 🛠️ Spesifikasi Teknologi
* Game Engine: Greenfoot IDE
* Bahasa Pemrograman: Java 
* Tipe Proyek: Game sederhana dengan konsep OOP

## 🚀 Cara Menjalankan Game

### Persiapan Awal
1. Pastikan komputer Anda sudah terinstal [Greenfoot IDE](https://greenfoot.org).
2. Unduh (download) atau lakukan clone repositori ini ke komputer Anda.

### Langkah Membuka Game
1. Buka aplikasi Greenfoot di laptop/komputer Anda.
2. Pilih menu Scenario pada bagian menu bar atas, lalu klik Open.
3. Cari dan pilih folder `game-cake-zayne` yang sudah diunduh tadi.
4. Klik tombol Run atau Act di bagian bawah jendela Greenfoot untuk mulai bermain.
